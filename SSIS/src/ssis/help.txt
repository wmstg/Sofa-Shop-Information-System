Sofa Shop Information Help

1. Use the Sofa Details panel to add new sofas to the table.

2. Enter a price and click on search to find a sofa for the given price. 
Only the first matched sofa will display.

3. Use the Sofa Inventory Search to display the number of sofas in the selected category.

4. Use the File -> Import Sofas, to import an XML file to populate the table with sofa details.
